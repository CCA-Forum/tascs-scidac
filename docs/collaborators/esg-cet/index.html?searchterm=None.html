ESG-CET: Earth System Grid Center for Enabling Technologies

by David E. Bernholdt — last modified Jun 01, 2008 05:08 PM
— filed under: SciDAC CET/Institute, TASCS Collaboration

Seeking opportunities to utilize component concepts and tools in the Earth System Grid's climate simulation data grid environment.

Collaboration Status: Awareness

TASCS Contact: David E. Bernholdt, ORNL, bernholdtde@ornl.gov

Collaboration Summary: Component concepts are being used in the architecture of the next generation of the Earth System Grid (ESG) software system.  We plan to introduce specific Common Component Architecture technologies as appropriate opportunities arise.

Collaboration Notes: Component concepts are being used in the architecture of the next generation of the Earth System Grid (ESG) software system. We are seeking opportunities to introduce specific Common Component Architecture tools and technologies, focusing on server-side analysis capabilities.  However, the climate modeling community already has a well-established collection of analysis tools which are not componentized and ESG's priorities are on supporting existing tools before developing new ones.  However a separate collaborative project (DMCP) is developing new data processing and analysis tools which are being implemented as components.
